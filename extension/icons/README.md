# Extension Icons

This directory should contain the following icon files:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Creating Icons

You can create simple icons using any graphics editor or online tool.

### Quick Option: Use an emoji or simple design

1. Use a service like https://favicon.io/ to generate icons
2. Create a simple PNG with your initials or a relevant symbol
3. Save in three sizes (16x16, 48x48, 128x128)

### Design Recommendations

- Use a simple, recognizable symbol (e.g., briefcase, clipboard, checkmark)
- High contrast colors for visibility
- Keep it simple - icons are small

## Temporary Workaround

Until proper icons are created, you can use placeholder images or the extension may show a default browser icon.
