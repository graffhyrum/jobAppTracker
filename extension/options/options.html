Get your API key from your tracker's settings page

How to Use

  1. Make sure your job tracker server is running
  2. Enter the API URL and API key above
  3. Navigate to a job posting page
  4. Click the extension icon in your browser toolbar
  5. Click "Extract from Page" to auto-fill job details
  6. Review and edit the information if needed
  7. Click "Save Application" to add it to your tracker

Supported Job Boards

  • LinkedIn Jobs
  • Indeed
  • Greenhouse
  • Lever
  • Generic job posting pages (best effort)
